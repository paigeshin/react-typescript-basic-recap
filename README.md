# Install React Typescript

```bash
npx create-react-app <appname> --template typescript
```

# Basic Setup

```tsx
import ReactDOM from "react-dom";

const App = () => {
  return (
    <div>
      <h1>Hi there!</h1>
    </div>
  );
};

ReactDOM.render(<App />, document.querySelector("#root"));
```
